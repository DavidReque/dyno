# Typography

El componente Typography es un elemento de interfaz que proporciona una forma consistente de mostrar texto con diferentes estilos, tamaños y pesos.

## Características

- Múltiples variantes (display, h1, h2, h3, h4, body1, body2, caption, overline)
- Diferentes pesos (regular, medium, semibold, bold)
- Diferentes colores (default, primary, secondary, success, error, warning)
- Elemento HTML personalizable (h1, h2, h3, h4, h5, h6, p, span)
- Diseño centrado por defecto

## Uso

```tsx
import { Typography } from "@your-package/components";

// Texto de visualización
<Typography variant="display">
  Texto de visualización
</Typography>

// Encabezado 1
<Typography variant="h1" weight="extrabold">
  Encabezado 1
</Typography>

// Encabezado 2
<Typography variant="h2" weight="semibold">
  Encabezado 2
</Typography>

// Cuerpo 1
<Typography variant="body1">
  Texto de cuerpo 1 para contenido principal. Este es un ejemplo más largo para mostrar cómo se ajusta el texto y cómo funciona la altura de línea.
</Typography>

// Cita
<Typography variant="blockquote">
  Texto de cita. Este es un ejemplo más largo para mostrar cómo se ajusta el texto y cómo funciona la altura de línea.
</Typography>

// Pie de texto
<Typography variant="caption">
  Texto de pie para detalles pequeños
</Typography>

// Código
<Typography variant="code" color="primary">
  pnpm install
</Typography>
```
