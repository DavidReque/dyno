# Typography

The Typography component is a UI element that provides a consistent way to display text with different styles, sizes, and weights.

## Features

- Multiple variants (display, h1, h2, h3, h4, body1, body2, caption, overline)
- Different weights (regular, medium, semibold, bold)
- Different colors (default, primary, secondary, success, error, warning)
- Customizable HTML element (h1, h2, h3, h4, h5, h6, p, span)
- Centered layout by default

## Usage

```tsx
import { Typography } from "@your-package/components";

// Display text
<Typography variant="display">
  Display text
</Typography>

// Heading 1
<Typography variant="h1" weight="extrabold">
  Heading 1
</Typography>

// Heading 2
<Typography variant="h2" weight="semibold">
  Heading 2
</Typography>

// Body 1
<Typography variant="body1">
  Body 1 text for main content. This is a longer example to show how text wraps and how line height works.
</Typography>

// Blockquote
<Typography variant="blockquote">
  Blockquote text. This is a longer example to show how text wraps and how line height works.
</Typography>

// Caption
<Typography variant="caption">
  Caption text for small details
</Typography>

// Code
<Typography variant="code" color="primary">
  pnpm install
</Typography>
```
