# Spinner

The Spinner component is an interface element that displays a rotating loading indicator to show that an action is being processed.

## Features

- Different sizes (sm, md, lg)
- Customizable color
- Customizable CSS classes
- Centered design by default

## Usage

```tsx
import { Spinner } from "@your-package/components";

// Basic Spinner
<Spinner />

// Spinner with custom size and color
<Spinner
  size="md"
  color="border-t-green-400"
/>

// Small Spinner
<Spinner size="sm" />

// Large Spinner
<Spinner size="lg" />
```
