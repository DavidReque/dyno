# Breadcrumb

The Breadcrumb component is a navigation element that displays the hierarchy of the current location.

## Features

- Hierarchical navigation
- Support for links
- Customizable via className
- Integration with the design system

## Usage

```tsx
import { Breadcrumb } from "@your-package/components";

// Basic Breadcrumb
<Breadcrumb
  items={[
    { label: "Home", href: "/" },
    { label: "Library", href: "/library" },
    { label: "Current Page" }
  ]}
/>

// Breadcrumb with custom items
<Breadcrumb
  items={[
    { label: "Dashboard", href: "/" },
    { label: "Projects", href: "/projects" },
    { label: "Project Alpha" }
  ]}
  className="custom-breadcrumb"
/>
```
