# Card

The Card component is a container that groups related content.

## Features

- Multiple variants (`default`, `ghost`, `outline`)
- Support for title, subtitle, and icon
- Customizable via `className`
- Integrated with the design system

## Usage

```tsx
import { Card } from "@your-package/components";
import { User } from "lucide-react";

// Basic card
<Card
  title="Card Title"
  subtitle="Card Subtitle"
>
  Card content goes here
</Card>

// Card with icon
<Card
  title="Card with Icon"
  subtitle="This card displays an icon"
  icon={User}
>
  The icon enhances the visual appeal
</Card>

// Card with footer
<Card
  title="Card with Footer"
  subtitle="This card includes a footer"
  footer="Learn More"
>
  Main content
</Card>
```
