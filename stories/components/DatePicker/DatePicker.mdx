# DatePicker

The DatePicker component allows users to select a date in an intuitive and visual way.

## Features

- Visual calendar interface
- Date selection via click
- Disabled state support
- CSS class customization
- Customizable labels
- Responsive design

## Usage

```tsx
import { DatePicker } from "@your-package/components";

const MyComponent = () => {
  return (
    <DatePicker label="Select Date" onChange={(date) => console.log(date)} />
  );
};
```
