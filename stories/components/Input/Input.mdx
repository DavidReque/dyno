# Input

The Input component is a text input field that allows users to enter data.

## Features

- Multiple variants (default, withButton, file, disabled, withLabel)
- Support for icons (leading and trailing)
- States (disabled, error)
- Customizable via className
- Integration with forms

## Usage

```tsx
import { Input } from "@your-package/components";
import { Search } from "lucide-react";

// Basic Input
<Input placeholder="Enter text..." />

// Input with icon
<Input
  leadingIcon={Search}
  placeholder="Search..."
  label="Search Input"
/>

// Input with button
<Input
  variant="withButton"
  placeholder="Enter email"
  onButtonClick={() => console.log("Button clicked")}
/>
```
