# Accordion

The Accordion component is a UI element that allows showing and hiding content in an organized way.

## Features

- Individual section expansion/collapse
- Support for custom content
- Responsive design
- Built-in accessibility

## Usage

```tsx
import { Accordion } from "@your-package/components";

const items = [
  {
    id: 1,
    title: "Section 1",
    content: "Section 1 content",
  },
  {
    id: 2,
    title: "Section 2",
    content: "Section 2 content",
  },
];

<Accordion items={items} />;
```
