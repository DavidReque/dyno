# Accordion

El componente Accordion es un elemento de interfaz que permite mostrar y ocultar contenido de forma organizada.

## Características

- Expansión/colapso de secciones individuales
- Soporte para contenido personalizado
- Diseño responsive
- Accesibilidad integrada

## Uso

```tsx
import { Accordion } from "@your-package/components";

const items = [
  {
    id: 1,
    title: "Sección 1",
    content: "Contenido de la sección 1",
  },
  {
    id: 2,
    title: "Sección 2",
    content: "Contenido de la sección 2",
  },
];

<Accordion items={items} />;
```
