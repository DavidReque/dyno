# Alert

The Alert component is a notification element that displays important messages to the user.

## Features

- Multiple variants (success, error, warning, info)
- Option to close (dismissible)
- Customizable via className
- Integration with the design system

## Usage

```tsx
import { Alert } from "@your-package/components";

// Basic alert
<Alert message="This is an alert message" />

// Alert with variant and option to close
<Alert
  variant="success"
  message="Operation completed successfully!"
  dismissible={true}
  onClose={() => console.log("Alert closed")}
/>

// Error alert
<Alert
  variant="error"
  message="An error occurred. Please try again."
  dismissible={true}
/>
```
