# Select

The Select component is a form element that allows users to select an option from a dropdown list.

## Features

- Multiple variants (primary, secondary, ghost, destructive)
- Different sizes (sm, md, lg)
- Disabled state
- Customizable options
- Customizable selected value
- Customizable placeholder text
- Customizable icon

## Usage

```tsx
import { Select } from "@your-package/components";

// Basic Select
<Select
  options={[
    { value: "1", label: "Option 1" },
    { value: "2", label: "Option 2" },
    { value: "3", label: "Option 3" }
  ]}
  placeholder="Select an option"
/>

// Select with variant and size
<Select
  options={[
    { value: "1", label: "Option 1" },
    { value: "2", label: "Option 2" },
    { value: "3", label: "Option 3" }
  ]}
  variant="primary"
  size="md"
  placeholder="Select an option"
/>

// Disabled Select
<Select
  options={[
    { value: "1", label: "Option 1" },
    { value: "2", label: "Option 2" },
    { value: "3", label: "Option 3" }
  ]}
  disabled={true}
  placeholder="Select an option"
/>
```
