# Checkbox

The Checkbox component is an input control that allows users to select an option.

## Features

- Selected/unselected state
- Support for labels
- Disabled state
- Customizable via className

## Usage

```tsx
import { Checkbox } from "@your-package/components";

// Basic Checkbox
<Checkbox
  label="Default Checkbox"
  defaultChecked={false}
/>

// Disabled Checkbox
<Checkbox
  label="Disabled Checkbox"
  defaultChecked={false}
  disabled={true}
/>

// Checkbox with onChange event
<Checkbox
  label="Checkbox with onChange"
  defaultChecked={false}
  onChange={(checked) => console.log("Checked:", checked)}
/>
```
