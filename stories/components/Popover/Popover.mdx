# Popover

El componente Popover es un elemento de interfaz que muestra contenido adicional cuando se activa, generalmente mediante un clic en un botón o elemento.

## Características

- Contenido personalizable
- Botón de activación personalizable
- Clases CSS personalizables para el botón y el panel
- Diseño centrado por defecto

## Uso

```tsx
import { Popover } from "@your-package/components";

// Popover básico
<Popover
  buttonContent="Open Popover"
  panelContent={<div>Contenido del popover</div>}
/>

// Popover con contenido personalizado
<Popover
  buttonContent="Opciones"
  panelContent={
    <div>
      <ul className="space-y-1">
        <li className="px-2 py-1 rounded">Opción 1</li>
        <li className="px-2 py-1 rounded">Opción 2</li>
        <li className="px-2 py-1 rounded">Opción 3</li>
      </ul>
    </div>
  }
/>
```
