# Popover

The Popover component is a UI element that displays additional content when triggered, usually by clicking on a button or element.

## Features

- Customizable content
- Customizable trigger button
- Customizable CSS classes for button and panel
- Centered layout by default

## Usage

```tsx
import { Popover } from "@your-package/components";

// Basic Popover
<Popover
  buttonContent="Open Popover"
  panelContent={<div>Popover content</div>}
/>

// Popover with custom content
<Popover
  buttonContent="Options"
  panelContent={
    <div>
      <ul className="space-y-1">
        <li className="px-2 py-1 rounded">Option 1</li>
        <li className="px-2 py-1 rounded">Option 2</li>
        <li className="px-2 py-1 rounded">Option 3</li>
      </ul>
    </div>
  }
/>
```
