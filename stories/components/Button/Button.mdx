# Button

El componente Button es un elemento fundamental de la interfaz que permite a los usuarios realizar acciones.

## Características

- Múltiples variantes (primary, secondary, ghost, destructive)
- Diferentes tamaños (sm, md, lg)
- Soporte para iconos
- Estados (disabled, loading)
- Personalizable mediante className

## Uso

```tsx
import { Button } from "@your-package/components";

// Botón básico
<Button>Click me</Button>

// Botón con variante y tamaño
<Button variant="primary" size="md">
  Submit
</Button>

// Botón con icono
<Button variant="secondary" icon={File}>
  Upload File
</Button>
```
