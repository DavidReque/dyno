# Button

The Button component is a fundamental interface element that allows users to perform actions.

## Features

- Multiple variants (primary, secondary, ghost, destructive)
- Different sizes (sm, md, lg)
- Support for icons
- States (disabled, loading)
- Customizable via className

## Usage

```tsx
import { Button } from "@your-package/components";

// Basic button
<Button>Click me</Button>

// Button with variant and size
<Button variant="primary" size="md">
  Submit
</Button>

// Button with icon
<Button variant="secondary" icon={File}>
  Upload File
</Button>
```
