# Skeleton

The Skeleton component is a UI element that displays a loading skeleton to indicate that content is loading.

## Features

- Multiple skeleton types (Card, Form, Code, Icon, List, Table, Paragraph)
- Customizable via CSS classes
- Centered layout by default
- Customizable count for lists

## Usage

```tsx
import {
  CardSkeleton,
  FormSkeleton,
  CodeSkeleton,
  IconSkeleton,
  ListSkeleton,
  TableSkeleton,
  ParagraphSkeleton
} from "@your-package/components";

// Card skeleton
<div className="w-80">
  <CardSkeleton />
</div>

// Form skeleton
<div className="w-80">
  <FormSkeleton />
</div>

// Code skeleton
<div className="w-80">
  <CodeSkeleton />
</div>

// Icon skeleton
<div className="flex items-center justify-center p-4">
  <IconSkeleton />
</div>

// List skeleton
<div className="w-80">
  <ListSkeleton count={5} />
</div>

// Table skeleton
<div className="w-full max-w-lg">
  <TableSkeleton rows={5} />
</div>

// Paragraph skeleton
<div className="w-80">
  <ParagraphSkeleton lines={4} />
</div>
```
