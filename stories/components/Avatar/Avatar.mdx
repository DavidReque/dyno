# Avatar

The Avatar component is a visual element that represents a user or entity.

## Features

- Multiple sizes (sm, md, lg)
- Supports images and fallback content
- Customizable via `className`
- Seamlessly integrates with the design system

## Usage

```tsx
import { Avatar } from "@your-package/components";

// Basic avatar with image
<Avatar
  src="https://example.com/avatar.jpg"
  alt="User Avatar"
  size="md"
/>

// Avatar with fallback
<Avatar
  alt="John Doe"
  size="lg"
  fallback="JD"
/>

// Small avatar
<Avatar
  src="https://example.com/avatar.jpg"
  alt="Small Avatar"
  size="sm"
/>
```
