# Tooltip

The Tooltip component is a UI element that displays additional information when the user hovers over an element.

## Features

- Customizable text
- Customizable position (top, bottom, left, right)
- Customizable CSS classes
- Centered layout by default

## Usage

```tsx
import { Tooltip } from "@your-package/components";

// Basic Tooltip
<Tooltip
  text="This is a tooltip"
  position="top"
>
  <span className="px-2 py-1 border rounded">Hover here</span>
</Tooltip>

// Bottom Tooltip
<Tooltip
  text="Bottom tooltip"
  position="bottom"
>
  <span className="px-2 py-1 border rounded">Hover here</span>
</Tooltip>

// Left Tooltip
<Tooltip
  text="Left tooltip"
  position="left"
>
  <span className="px-2 py-1 border rounded">Hover here</span>
</Tooltip>

// Right Tooltip
<Tooltip
  text="Right tooltip"
  position="right"
>
  <span className="px-2 py-1 border rounded">Hover here</span>
</Tooltip>
```
