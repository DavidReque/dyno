# RadioButton

The RadioButton component is a form element that allows users to select one option from a set of mutually exclusive options.

## Features

- Customizable label
- Customizable value
- Default selected state
- Disabled state
- Customizable CSS classes
- onChange event to handle selection

## Usage

```tsx
import { RadioButton } from "@your-package/components";

// Basic RadioButton
<RadioButton
  label="Default option"
  value="default"
/>

// Disabled RadioButton
<RadioButton
  label="Disabled option"
  value="disabled"
  disabled={true}
/>

// RadioButton with initial selected state
<RadioButton
  label="Preselected option"
  value="preselected"
  defaultChecked={true}
/>
```
