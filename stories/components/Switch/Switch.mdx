# Switch

The Switch component is a UI element that allows users to toggle between two states, usually on and off.

## Features

- Controlled state via checked and onChange
- Centered layout by default
- Automatic documentation

## Usage

```tsx
import { Switch } from "@your-package/components";
import { useState } from "react";

// Basic Switch with controlled state
const MyComponent = () => {
  const [value, setValue] = useState(false);
  return <Switch checked={value} onChange={setValue} />;
};

// Switch with initial state
const MyComponentWithInitialState = () => {
  const [value, setValue] = useState(true);
  return <Switch checked={value} onChange={setValue} />;
};
```
