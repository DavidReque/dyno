# Tabs

The Tabs component is a UI element that allows users to navigate between different sections of content without changing pages.

## Features

- Customizable tabs with labels
- Customizable content for each tab
- Support for icons in tabs
- Automatic documentation

## Usage

```tsx
import Tabs from "@your-package/components";
import { LucideRefreshCw } from "lucide-react";

// Basic Tabs
<Tabs
  tabs={[
    { id: "tab1", label: "Tab 1", content: <p>Tab 1 Content</p> },
    { id: "tab2", label: "Tab 2", content: <p>Tab 2 Content</p> },
    { id: "tab3", label: "Tab 3", content: <p>Tab 3 Content</p> }
  ]}
/>

// Tabs with icon
<Tabs
  tabs={[
    { id: "tab1", label: "Tab 1", content: <p>Tab 1 Content</p> },
    {
      id: "tab2",
      label: "Tab 2 with icon",
      content: <p>Tab 2 Content</p>,
      icon: <LucideRefreshCw size={16} />
    },
    { id: "tab3", label: "Tab 3", content: <p>Tab 3 Content</p> }
  ]}
/>
```
