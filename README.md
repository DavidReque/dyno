## Installation

```bash
npm install @dyno/ui
# or
yarn add @dyno/ui
```

## Quick Start

```tsx
import { Button, Modal, DatePicker } from "@dyno/ui";

function App() {
  return (
    <div>
      <Button variant="primary">Click me</Button>
      <DatePicker label="Select date" />
      <Modal title="Hello" isOpen={true}>
        Content
      </Modal>
    </div>
  );
}
```

## Development

### Setup

1. Clone the repository:

```bash
git clone https://github.com/your-org/dyno-ui.git
cd dyno-ui
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start Storybook:

```bash
npm run storybook
# or
yarn storybook
```


## Testing

```bash
# Run tests
npm test
# or
yarn test

# Run tests with coverage
npm run test:coverage
# or
yarn test:coverage
```

## Building

```bash
# Build the library
npm run build
# or
yarn build
```
